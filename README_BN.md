# 🚀 WinSetup: আল্টিমেট উইন্ডোজ পোস্ট-ইনস্টলেশন টুল

**WinSetup** হলো একটি শক্তিশালী, আধুনিক এবং ওপেন-সোর্স সফটওয়্যার ইনস্টলার যা
উইন্ডোজ ১০ এবং ১১ এর জন্য তৈরি করা হয়েছে। এটি **PowerShell** এবং **Microsoft
Winget** এর সমন্বয়ে কাজ করে, যার মাধ্যমে ব্যবহারকারী মাত্র কয়েক ক্লিকে ১০০টিরও
বেশি প্রয়োজনীয় সফটওয়্যার তাদের নতুন পিসিতে ইনস্টল করতে পারবেন।

## ✨ মূল ফিচারসমূহ

- **১০০+ সফটওয়্যার কালেকশন:** ব্রাউজার থেকে শুরু করে ডেভেলপার টুলস—সবচেয়ে বেশি
  ব্যবহৃত সব সফটওয়্যার এখানে যুক্ত করা হয়েছে।
- **আধুনিক ডার্ক ইউআই (Dark UI):** VS Code থেকে অনুপ্রাণিত একটি চমৎকার এবং
  ব্যবহারবান্ধব ইন্টারফেস।
- **ক্যাটাগরি ভিত্তিক সিলেকশন:** আপনার প্রয়োজন অনুযায়ী সহজে ক্যাটাগরি (Media,
  Dev, Gaming, ইত্যাদি) থেকে অ্যাপ বেছে নিতে পারবেন।
- **এক ক্লিকে ইনস্টলেশন:** একসাথে অনেকগুলো অ্যাপ সিলেক্ট করুন এবং এক ক্লিকেই
  ব্যাকগ্রাউন্ডে ইনস্টল করুন (Silent Install)।
- **রিয়েল-টাইম প্রগ্রেস ও লগ:** ইনস্টলেশন কতটুকু হলো তা দেখার জন্য লাইভ
  প্রগ্রেস বার এবং টার্মিনাল-স্টাইল লগ উইন্ডো।
- **উইংগেট (Winget) ইন্টিগ্রেশন:** নিরাপদ এবং ভেরিফাইড ডাউনলোডের জন্য অফিসিয়াল
  মাইক্রোসফট প্যাকেজ ম্যানেজার ব্যবহার করা হয়েছে।

## 🛠️ কিভাবে ব্যবহার করবেন

আপনার পিসি সেটআপ করা এখন আগের চেয়েও সহজ:

১. এই রিপোজিটরি (Repository) আপনার কম্পিউটারে **ডাউনলোড** করুন। ২. ফোল্ডারে থাকা
`start.bat` ফাইলটি খুঁজে বের করুন। ৩. `start.bat` ফাইলটির ওপর **রাইট ক্লিক**
করুন এবং **"Run as Administrator"** সিলেক্ট করুন। ৪. **WinSetup** ইন্টারফেসটি
ওপেন হবে। ৫. আপনার পছন্দের **Category** সিলেক্ট করুন, অ্যাপগুলোতে টিক দিন এবং
**INSTALL SELECTED** বাটনে ক্লিক করুন।

> **দ্রষ্টব্য:** সফটওয়্যারগুলো সিস্টেম-ওয়াইড ইনস্টল করার জন্য
> 'অ্যাডমিনিস্ট্রেটর' পারমিশন প্রয়োজন।
